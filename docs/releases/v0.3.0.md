# v0.3.0 (8th January 2026)

## Highlights

* Starting ith this release, checkmake is no longer considered experimental.
* The container images on [docker hub](https://hub.docker.com/r/mrtazz/checkmake/) are discontinued ; images are nowpublished on [quay.io](https://quay.io/repository/checkmake/checkmake) instead.
* In addition to the previously published build assets, an [arm64 binary for Linux](https://github.com/checkmake/checkmake/releases/download/v0.3.0/checkmake-v0.3.0.linux.arm64) is now published.
